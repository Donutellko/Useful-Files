Тайловый оконный менеджер.
Документация:
  https://i3wm.org/docs/userguide.html
Краткий гайд для начала со шпаргалкой по шорткатам и настройкой работы на двух мониторах:
  https://habrahabr.ru/post/150100/
Видео с GoogleTechTalks:
  http://www.youtube.com/watch?v=QnYN2CTb1hM

Установка:
apt-get install i3

cp /etc/i3/config ~/.i3/config # применение дефолтного конфига

В качестве клавиши-модификатора можно выбрать Win или Alt. В этом файле шорткаты приведены с модификатором Win


Win+Enter открывает терминал
Win+d запускает dmenu (меню сверху экрана, которое по мере ввода с клавиатуры названия приложения предлагает варианты для запуска)
Win+Shift+Q закрывает активное окно

Win+v включает режим вертикального тайлинга (экран будет делиться горизонтально)
Win+h включает режим горизонтального тайлинга (экран будет делиться вертикально)

Win+w включает режим вкладок (каждое окно на рабочем столе занимает весь экран, сверху видны вкладки)
Win+s включает стековый режим (заголовки окон один под другим, каждое окно занимает весь экран)
Win+e возвращает стандартный режим
Win+Shift+Space переключает окно в режим плавающего и обратно

Win+Left/Right/Up/Down перемещает фокус в пределах рабочего стола
Win+Shift+Left/Right/Up/Down перемещает текущее окно в пределах рабочего стола

Win+1 и т.д. переключает на рабочий стол с указанным номером

Win+Shift+C читает настройки из файла конфигурации
Win+Shift+E выходит из i3wm на экран ввода имени пользователя и пароля
